    Habitica Dailies Inequality

This project collects structured evidence for analysing how Habitica's Dailies system translates interrupted participation into measurable loss.

    Project structure

- `00_evidence/screenshots/` - interface screenshots
- `00_evidence/notes/` - screenshot notes
- `00_evidence/data/` - structured CSV evidence
- `scripts/` - Python scripts for creating and validating CSV
- `index.html` - simple evidence gallery
- `style.css` - gallery styling
- `script.js` - screenshot rendering logic
